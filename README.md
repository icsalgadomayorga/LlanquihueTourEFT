## 📘 Descripción general del sistema
Este proyecto corresponde a la Evaluación Final Transversal del ramo Desarrollo Orientado a Objetos I. Se trata de un sistema orientado a objetos desarrollado en Java, cuyo objetivo es solucionar la problemática presentada por la empresa ficticia Llanquihue Tours.

## ⚙️ Instrucciones para clonar y ejecutar el proyecto

1. Clona el repositorio desde GitHub:

```bash
git clone https://github.com/icsalgadomayorga/LlanquihueTourEFT.git
```

2. Abre el proyecto en IntelliJ IDEA.

3. Ejecuta el archivo `Main.java` desde el paquete `app`.

4. Agrega o elimina Clientes, Empleados, Proveedores o Productos según las necesidades de la empresa
